SOSFramework String Constants

List of string constants associated with the Service SDK SOS framework.

  • ServiceCloud.SOS.Title

    Description

    Dialogs */ /* The title we present for any UI element that has a title, such as one of our dialogs

    Default Value

    Salesforce SOS

  • ServiceCloud.SOS.ConnectPrompt

    Description

    The dialog body we present when prompting to start an SOS session

    Default Value

    You are about to start an SOS session. You will be able to talk and screen share with an Agent in real time. Are you ready to experience the future of customer service?

  • ServiceCloud.SOS.Session.Disconnect

    Description

    The dialog body we present when confirming that a user would like to end their session (after hitting an end session button)

    Default Value

    Press OK to disconnect from the session.

    Description

    The dialog body we present when the agent has ended the SOS session

    Default Value

    The Agent has ended the SOS session.

    Description

    The dialog body we present when the agent disconnects for an unknown reason

    Default Value

    The agent may have disconnected. Press Quit to end the session now or Cancel to continue waiting.

    Description

    The dialog body we present when asking a customer if they would like to continue waiting for a session

    Default Value

    All agents are currently busy, do you wish to continue waiting?

    Description

    The dialog body we present when we cannot start a session due to no agents being online"

    Default Value

    Your SOS Session has timed out. Please try again.

    Description

    The dialog body we present when we cannot start a session due to insufficient network conditions
