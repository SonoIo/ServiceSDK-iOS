Reference KnowledgeFramework String Constants

KnowledgeFramework String Constants

List of string constants associated with the Service SDK Knowledge framework.

  • ServiceCloud.DeveloperUtility.VersionLabel

    Description

    Development status message

    Default Value

    v%@

  • ServiceCloud.DeveloperUtility.Title

    Description

    Development title label

    Default Value

    Service Cloud SDK

  • ServiceCloud.DeveloperUtility.Network.Title

    Description

    Development network section title

    Default Value

    Networking

  • ServiceCloud.DeveloperUtility.Data.Title

    Description

    Development Data section title

    Default Value

    Data

  • ServiceCloud.DeveloperUtility.Network.Prompt

    Description

    Development network section title

    Default Value

    Changes take effect after reopening the service UI

  • ServiceCloud.DeveloperUtility.Network.MockData

    Description

    Development network mock data title

    Default Value

    Use mock network data

  • ServiceCloud.DeveloperUtility.Data.ClearCache

    Description

    Development data clear cache title

    Default Value

    Clear Cache

  • ServiceCloud.DeveloperUtility.Data.ClearCacheAlertMessage

    Description

    Development clear cache alert message

    Default Value

    Are you sure you want to clear the cache?

  • ServiceCloud.DeveloperUtility.Data.CancelButtonTitle

    Description

    Development clear cache alert cancel button title

    Default Value

    Cancel

  • ServiceCloud.DeveloperUtility.Data.OKButtonTitle

    Description

    Development clear cache alert ok button title

    Default Value

    OK

  • ServiceCloud.ServiceInterface.ClearCacheAlertMessage

    Description

    Alert message when clearcache is set and help is opened

    Default Value

    Please restart the app to clear out cache

  • ServiceCloud.ServiceInterface.ClearCacheAlertOKButtonTitle

    Description

    Development clear cache alert ok button title

    Default Value

    OK

  • ServiceCloud.SupportHome.EmptyArticleListMsgTitle

    Description

    Main message displayed when the articlelist is empty in support home

    Default Value

    Well, this is embarrassing

  • ServiceCloud.SupportHome.EmptyArticleListMsgDescription

    Description

    Second line of the message displayed when article list is empty in Support home

    Default Value

    There are no articles in this category

  • ServiceCloud.SupportHome.ExpandCategory

    Description

    Accessibility message to expand category

    Default Value

    Expand category %@

  • ServiceCloud.SupportHome.CollapseCategory

    Description

    Accessibility message to collapse category

    Default Value

    Collapse category %@

  • ServiceCloud.SupportHome.HelpTitle

    Description

    Title of a support home controller

    Default Value

    HELP

  • ServiceCloud.SupportHome.Close

    Description

    Accessibility message of the close button

    Default Value

    Close

  • ServiceCloud.SupportHome.InvalidDataCategoryOrGroupErrorMsgTitle

    Description

    Error message title displayed for invalid data category group

    Default Value

    Invalid data category group or data category selected.

  • ServiceCloud.SupportHome.InvalidDataCategoryOrGroupErrorMsgDesc

    Description

    Error message description displayed for invalid data category

    Default Value

    Please contact the administrator.

  • ServiceCloud.ArticleDetail.Back

    Description

    Accessibility message of the back button on Article Detail

    Default Value

    Back

  • ServiceCloud.ArticleDetail.HelpTitle

    Description

    Title of a article detail controller

    Default Value

    HELP

  • ServiceCloud.ArticleDetail.EmptyMsg

    Description

    Message to display when no article has yet been selected

    Default Value

    Select an article or category to explore solutions

  • ServiceCloud.ArticleDetail.MinimizeButton

    Description

    Accessibility Message of the minimize button on Article Detail

    Default Value

    Minimize

  • ServiceCloud.DataCategoryDetail.NoDataMsgTitle

    Description

    Title to display when Data Category Detail View is empty

    Default Value

    This category is empty

  • ServiceCloud.DataCategoryDetail.NoDataMsgDescription

    Description

    Description to display when Data Category Detail View is empty

    Default Value

    There are no articles in this category

  • ServiceCloud.DataCategoryDetail.HelpTitle

    Description

    Title of a data category detail controller

    Default Value

    HELP

  • ServiceCloud.DataCategoryDetail.SubcategoryHeader

    Description

    Title of the Sub category header

    Default Value

    Sub Categories

  • ServiceCloud.OverlayPresentation.Close

    Description

    Accessibility label for the overlay controller

    Default Value

    Close

  • ServiceCloud.ShowMoreCellView.ShowMoreLabelTitle

    Description

    Label for ShowMoreCell title

    Default Value

    Show More

  • ServiceCloud.ShowMoreCellView.ErrorMessage

    Description

    Label for ShowMoreCell title

    Default Value

    Unable to load more articles...

  • ServiceCloud.DataCategoryDetail.NoSearchResultMsgTitle

    Description

    Title to display when no search results are found

    Default Value

    No results found

  • ServiceCloud.DataCategoryDetail.NoSearchResultMsgDescription

    Description

    Description to display when no search results are found

    Default Value

    Please make sure your words are spelled correctly, or use different keywords.

  • ServiceCloud.EmptySearchView.TitleMessage

    Description

    Empty search page title string

    Default Value

    Help Search

  • ServiceCloud.EmptySearchView.DescriptionMessage

    Description

    Empty search page description string

    Default Value

    Search our help articles and solutions
