CasesFramework String Constants

List of string constants associated with the Service SDK Cases framework.

  • ServiceCloud.CasePublisher.Title

    Description

    Title of the Case Publisher controller

    Default Value

    CONTACT

  • ServiceCloud.CasePublisher.Close

    Description

    Accessibility label for the close button on Case Publisher

    Default Value

    Close

    Label of the submit button on Case Publisher

    Default Value

    Send

    Description

    Title of the alert shown on closing case publisher

    Default Value

    Save Your Message?

    Description

    Default Value

    Save

    Description

    Label of the discard action on alert shown when closing case publisher

    Default Value

    Discard

    Default Value

    Description

    Accessibility Label of the required field on Case Publisher

    Default Value

    CASE CREATED
